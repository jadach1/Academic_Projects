On linux this requires a -std=c++11 or higher flag to be attached to the compiler

The program takes 6 arguments, a file name and a parser for example: file1 , file2 , file3 , OR name1 | name2 | name3|

Simple factory simulation, takes a list of tasks, customer orders and items.  The factory attempts to fill each customer order with the tasks and items on hand.

Uses Graphviz to visually represent the flow of the order, examples can be seen in the files above

You will need to download graphviz in order for this program to work
